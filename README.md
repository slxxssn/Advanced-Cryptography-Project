Project Title

Secure File Encryption System

Project Overview

This project demonstrates the implementation of various cryptographic techniques including classical encryption, stream ciphers, block ciphers, and public key cryptography. The system was developed using Python to provide practical understanding of modern cryptographic concepts and secure communication techniques.

Project Objectives

* Understand cryptographic principles.
* Implement encryption and decryption algorithms.
* Perform randomness testing.
* Secure files using AES encryption.
* Implement RSA public key cryptography.
* Apply secure coding and version control practices.
